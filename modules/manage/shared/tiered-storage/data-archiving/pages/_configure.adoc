Data archiving requires a Tiered Storage configuration.
:description: 

. xref::tiered-storage.adoc#set-up-tiered-storage[Set up Tiered Storage] for the cluster or for specific topics.
. If you want to read data from object storage, xref::tiered-storage.adoc#remote-read[enable remote read]. When remote read is disabled, data can only be read from local storage.
. xref::tiered-storage.adoc#set-retention-limits[Set retention limits].

To recover a topic from object storage, use xref::tiered-storage.adoc#remote-recovery[remote recovery].
