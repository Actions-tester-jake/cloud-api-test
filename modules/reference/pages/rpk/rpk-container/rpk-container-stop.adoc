= rpk container stop
:description: rpk container stop

Stop an existing local container cluster.

== Usage

[,bash]
----
rpk container stop [flags]
----

== Flags

[cols=",,",]
|===
|*Value* |*Type* |*Description*
|-h, --help |- |Help for stop.
|-v, --verbose |- |Enable verbose logging (default `false`).
|===
